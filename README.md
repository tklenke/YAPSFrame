# YAPSFrame
Yet Another (Raspberry) Pi Smart Frame

##Dependencies

```
sudo apt-get install python-tk
sudo apt-get install python3-tk
sudo apt-get install python3-pil.imagetk
sudo apt-get install python-imaging-tk
sudo pip install config
```
